# کلمه‌سازی با جاوا

## مشخصات پروژه
- **عنوان پروژه:** کلمه‌سازی با جاوا  
- **نوع پروژه:** پروژه پایان‌ترم  
- **درس:** توسعه موبایل  
- **استاد:** محمدخانی  
- **زبان برنامه‌نویسی:** Java  

## توضیح پروژه
این پروژه یک بازی ساده مبتنی بر کلمات است که با زبان برنامه‌نویسی Java توسعه داده شده است.  
در این بازی، کاربر با استفاده از حروف داده‌شده اقدام به ساخت کلمات می‌کند و منطق بازی بر اساس بررسی صحت و امتیازدهی به کلمات پیاده‌سازی شده است.

## نحوه اجرا
این پروژه با استفاده از **IntelliJ IDEA** اجرا می‌شود.  
برای اجرای برنامه، کافی است کلاس `Main` را Run کنید.
